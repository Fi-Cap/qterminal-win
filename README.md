# QTerminal-Win
 
A Windows port of the Qt-based terminal emulator QTerminal, with no forced Git-Bash dependency. Supports custom shell (path/arguments) via config file, retaining tabbed interface and core features for flexible Windows workflows.
 
## Key Features
 
- Windows-native port of QTerminal’s core functionality
 
## Origin
 
Forked from lxqt/qterminal — built for Windows users seeking a flexible, Qt-powered terminal.
